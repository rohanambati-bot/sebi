Vector: Email / SMS / WhatsApp / Social

Phishing & Impersonation Scanner

Paste a message an investor received. The engine scores it against known phishing signatures: urgency pressure, credential harvesting, unrealistic-return scam language, look-alike domains, and sender spoofing — and explains every flag it raises, so investors learn to spot the next one themselves.

Closing the trust gap

Authentic Communication Verifier

Issuers (SEBI, exchanges, brokers, listed companies) register genuine outgoing communications here, receiving a short VERIFY-CODE to print on the real message. Investors paste a code — or the full message text — and get an instant authenticity verdict, including detection of copy-pasted-and-edited forwards.

Vector: Deepfake images / manipulated visuals

Synthetic Media Forensics

Upload an image (e.g. a screenshot of a "CEO announcement" or executive photo circulating on social media). The scanner runs Error Level Analysis — a real forensic compression-artifact technique — plus EXIF metadata checks, and flags likely manipulation with a visual evidence map.

Scope & honesty note

What's real in this build, what's roadmap

Fully working now: the phishing/impersonation heuristic engine, the authenticity registry (register + verify by code + fuzzy content match), and the image forensics module (genuine Error Level Analysis + EXIF checks) — all running live against the FastAPI backend, no mocked responses.

Round-2 roadmap: voice-call spoof detection (wav2vec2 / ASVspoof-style acoustic model), video deepfake detection (frame-level CNN classifier, e.g. fine-tuned on FaceForensics++), a persistent database + admin console for issuers instead of the in-memory demo registry, and a Chrome/WhatsApp-adjacent browser extension so investors don't have to copy-paste manually.

See README.md and docs/EVALUATION.md in the project files for architecture, target users, channels, and detection performance on the sample dataset.
